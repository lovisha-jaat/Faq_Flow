# FAQFlow AI

FAQFlow AI is a multi-company FAQ management and intelligent chatbot platform built using Flask, Python, HTML, CSS, JavaScript and SQLite.

## Main Features

- Company registration and login
- Separate FAQ knowledge base for every company
- Add, edit and delete FAQs
- FAQ chatbot
- Visitor query history
- Dashboard statistics
- Chatbot preview
- Website integration widget
- AI-powered answers in the advanced version

## Technology Stack

- Python
- Flask
- SQLite
- HTML
- CSS
- JavaScript
